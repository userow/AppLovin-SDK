## Installation

AppLovin-SDK is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'AppLovin-SDK', :git=> 'https://github.com/userow/AppLovin-SDK.git', :tag=> '4.3.0'
```
